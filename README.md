# gulp-spritesmith-sample

```
.
├── README.md
├── dev
│   ├── scss
│   │   ├── sprite
│   │   │   └── _sample.scss
│   │   └── style.scss
│   └── sprite
│       └── sample
│           ├── icon1.png
│           ├── icon2.png
│           └── icon3.png
├── gulpfile.js
├── node_modules
├── package.json
└── pub
    ├── css
    │   └── style.css
    ├── image
    │   └── sprite
    │       └── sample.png
    └── index.html
```
